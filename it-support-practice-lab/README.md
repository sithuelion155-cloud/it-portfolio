# IT Support Practice Lab

This project documents simulated IT support scenarios to practice real-world troubleshooting and user support.

---

## Issue 1: Wi-Fi Connected but No Internet

**Problem**
User device shows Wi-Fi connection but cannot access websites or online services.

**Troubleshooting Steps**
1. Checked network connection status.
2. Restarted Wi-Fi adapter.
3. Flushed DNS cache using command prompt.
4. Restarted router.

**Resolution**
Resetting the network adapter and flushing DNS restored internet access.

**Skills Practiced**
- Basic network troubleshooting
- Logical problem-solving
- Clear issue documentation
